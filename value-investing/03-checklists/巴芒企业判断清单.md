# 巴芒企业判断清单

> 用途：每次研究一家企业时，先完成这张清单，再决定是否值得进入案例层或估值层。

## 1. 能力圈

| 问题 | 记录 |
|---|---|
| 我能否用 3 句话说清楚公司靠什么赚钱？ |  |
| 我能否列出收入、利润率、资本开支的前三个变量？ |  |
| 我知道哪些问题自己还不懂吗？ |  |
| 哪些信息会证明我理解错了？ |  |

## 2. 生意质量

| 问题 | 记录 |
|---|---|
| 这家公司是否能长期产生现金流？ |  |
| 增长是否需要大量资本投入？ |  |
| 新增资本回报率是否可能下降？ |  |
| 周期下行时，它是被动承压还是能扩大优势？ |  |

## 3. 护城河

| 问题 | 记录 |
|---|---|
| 护城河来自无形资产、转换成本、网络效应、规模经济，还是其他来源？ |  |
| 护城河是否能转化为定价权、成本优势或更高资本回报？ |  |
| 竞争对手最可能从哪里突破？ |  |
| 护城河是变深、稳定，还是正在变浅？ |  |

## 4. 管理层与资本配置

| 问题 | 记录 |
|---|---|
| 管理层是否有长期主义和纪律性？ |  |
| CAPEX、并购、分红、回购的优先级是否合理？ |  |
| 管理层是否愿意报告坏消息？ |  |
| 是否存在为了增长而牺牲回报率的风险？ |  |

## 5. 安全边际

| 问题 | 记录 |
|---|---|
| 这是好企业，还是好投资？两者是否被混淆？ |  |
| 当前价格隐含了什么增长和利润率假设？ |  |
| 如果增长、利润率或估值下修 30%，结论是否仍成立？ |  |
| 最大永久性损失来自哪里？ |  |

## 6. 结论分级

| 等级 | 含义 |
|---|---|
| A | 能力圈内，高质量企业，估值和风险仍需单独判断 |
| B | 企业质量较好，但关键变量仍需验证 |
| C | 资料不足，暂时只跟踪，不进入投资判断 |
| D | 超出能力圈或反方证据过强，暂停 |
