## 使用规则

1. 新资料先进入来源层或既有 `02-kb/sources`，不要直接扩写成观点。
2. 只有能改变概念理解、判断清单或案例结论的内容，才进入本目录。
3. 每个案例都必须保留反方证据和待验证项，避免把优质企业直接等同为优质投资。
