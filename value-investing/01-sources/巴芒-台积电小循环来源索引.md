# 巴芒-台积电小循环来源索引

## 本轮目标

用少量高质量来源完成一次小循环：

```text
巴芒原则 -> 概念卡 -> 判断清单 -> 03 科技时代的价值投资 -> 台积电案例卡 -> 复盘
```

本轮不重新导入大量原始资料，只复用现有知识库中已经整理过的来源和实体档案。

## 本轮不纳入

- 不直接展开台积电估值模型。
- 不扩展到所有半导体公司。
- 不把 AI CAPEX 景气判断直接等同为价值投资结论。
- 不把巴菲特曾买过台积电作为独立投资理由，只作为案例背景。

## 资料使用原则

1. 价值投资来源负责回答：什么是好企业、好价格和长期风险。
2. 半导体来源负责回答：台积电的业务变量、行业结构和证伪信号。
3. 台积电作为 `00-framework/案例卡-台积电.md`，由 03 主题页在文末链接进入，不改变主题主架构。
4. 吴梓豪模型作为高置信度外部预测锚点使用；实际财报、法说会和产能数据仍是最终验证依据。
