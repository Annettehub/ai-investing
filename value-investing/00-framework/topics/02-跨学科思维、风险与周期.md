# 02 跨学科思维、风险与周期

## 话题定位

这是价值投资知识库的决策层，解决的不是“什么是好企业”，而是：

> 当信息不完整、未来不可预测、市场情绪极端或系统发生变化时，如何降低判断错误和本金永久损失的概率？

它是巴芒原则与科技投资之间的桥梁。芒格提供多元模型，塔勒布提供不对称风险视角，霍华德·马克斯提供风险与周期判断，布莱恩·阿瑟提供复杂系统和报酬递增视角。

## 三组模型

### 1. 认知与决策

- 多元思维模型：用经济学、心理学、数学、工程学和生物学等多个视角检查同一问题。
- 逆向思考：先问“怎样会失败”，再问“怎样成功”。
- 第二层思维：不只判断事实，还判断市场已经定价了什么。
- 反偏见：识别确认偏误、易得性偏差、自视过高、叙事偏误和损失厌恶。

### 2. 风险与不确定性

- 风险不等同于波动率，重点是本金永久损失。
- 杠铃结构和凸性：让小部分风险暴露换取大上行，同时保护大部分本金。
- 不做精确预测，先准备多个情景和应对方案。
- 判断自己是否有足够的时间、现金流和仓位承受“过早正确”。

### 3. 周期与复杂系统

- 市场情绪和资本周期会在极端之间摆动，合理状态并不常见。
- 报酬递增、网络效应和路径依赖可能让早期优势不断扩大。
- 复杂系统不能简单用线性外推解释，局部变量变化可能引发系统性结果。
- 技术革命往往经历基础设施建设、资本狂热、制度调整和成熟阶段。

## 决策清单

1. 我真正理解的变量是什么？哪些只是叙事？
2. 当前价格隐含了什么增长、利润率和竞争格局？
3. 最坏情况下是波动，还是本金永久损失？
4. 这是短期噪声、正常周期，还是商业模式发生了变化？
5. 哪个事实出现后，我必须承认原判断错误？
6. 我的仓位和时间是否允许判断暂时不被市场验证？

## 使用边界

这些模型用于改进判断和风险管理，不用于把任何复杂现象强行解释成一个漂亮的理论。每个模型都要注明适用条件、反例和不能解释的部分。
