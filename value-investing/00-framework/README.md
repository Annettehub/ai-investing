# 价值投资框架

> 定位：独立的价值投资认知系统，与 AI 产业投研并行运行，互不干扰。
> 核心：收集、整理、内化巴菲特/芒格/格雷厄姆等价值投资经典理论，以及商业探索的底层逻辑。
> 边界：不参与 AI 产业投资判断，不输出交易建议。只负责建立扎实的认知底盘。

## 信息纳入流程

### 每日 SOP（与 AI 投研系统独立运行）

```
Step 1: 扔
→ 资料扔进 05-raw/inbox/ 或 IMA "价值投资"知识库

Step 2: 跑同步（人工触发）
→ 对 WorkBuddy 说："跑价值投资 ingest"
→ AI 拉取 → 生成摘要 → 标记关联

Step 3: 用户确认分发
→ 确认后写入 02-concepts/ 或 03-mental-models/ 或 04-case-studies/

Step 4: 更新索引
→ 更新 00-framework/index.md（如有）
```

### 资料分类标准

| 资料类型 | 存放位置 | 示例 |
|---------|---------|------|
| 巴菲特/芒格言论 | 01-sources/ + 关联概念 | 股东信、采访、股东大会 |
| 投资原则/定义 | 02-concepts/ | 能力圈、护城河、安全边际 |
| 多学科思维模型 | 03-mental-models/ | 复利、逆向思维、机会成本 |
| 具体公司分析 | 04-case-studies/ | 喜诗糖果、可口可乐、苹果 |
| 商业底层逻辑 | 02-concepts/ 或 03-mental-models/ | 品牌溢价、网络效应、规模经济 |

## 与 AI 投研系统的边界

| 维度 | 价值投资框架 | AI 投研系统 |
|------|-------------|------------|
| **目标** | 建立认知底盘，内化判断框架 | 追踪产业动态，输出交易信号 |
| **时间尺度** | 10年+，跨越周期 | 1-3年，季度跟踪 |
| **信息类型** | 经典理论、商业本质、人性 | 技术路线、产能数据、政策 |
| **输出** | 概念卡片、思维模型、案例库 | 假设、评级、买卖建议 |
| **是否参与交易** | ❌ 绝不 | ✅ 核心目标 |

**交叉点**：当评估一家 AI 公司的长期护城河或管理层质量时，可以引用价值投资框架的概念，但价值投资框架本身不生成 AI 标的的买卖判断。
