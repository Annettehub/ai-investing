# 护城河

## 定义

护城河是企业长期维持高资本回报的结构性原因，不是单纯的技术领先、品牌知名度或短期业绩增长。

有效护城河至少需要同时回答三个问题：

1. 为什么客户继续选择它？
2. 为什么竞争对手难以复制？
3. 为什么这种优势能转化为更高、更稳定或更持久的资本回报？

## 常见类型

| 类型 | 判断重点 |
|---|---|
| 无形资产 | 品牌、专利、标准、认证是否带来定价权或准入壁垒 |
| 转换成本 | 客户迁移是否带来良率、设计、时间、供应链和组织成本 |
| 网络效应 | 使用者或生态参与者越多，产品/平台越强 |
| 规模经济 | 固定成本、研发、供应链和产能利用率是否随规模摊薄 |

## 台积电应用

台积电的护城河不是单点技术领先，而是多层叠加：

| 护城河来源 | 台积电表现 | 需要继续验证 |
|---|---|---|
| 先进制程 | 3nm/2nm 节点领先，先进客户集中 | 三星、英特尔是否拿到关键大客户 |
| 先进封装 | CoWoS/SoIC/SoW-X 与 AI GPU/ASIC 绑定 | CoWoS 扩产是否过剩，竞争者是否追上 |
| 客户转换成本 | 苹果、NVIDIA、AMD、博通等客户迁移成本极高 | 客户是否因价格、地缘或供应安全分散订单 |
| 规模经济 | 高市占率摊薄研发、设备和工艺学习成本 | 过高 CAPEX 是否削弱回报率 |
| 生态标准 | OIP、3D IC 设计标准和供应链协同 | 标准是否被替代或分裂 |

## 易错点

- 技术领先不自动等于护城河，必须看领先能否持续变现。
- 护城河深不等于价格永远合理。
- 高 CAPEX 行业里，增长可能伴随折旧和投入产出比压力。
