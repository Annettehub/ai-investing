# 资本配置与复利

## 定义

复利不是利润自然增长，而是企业能长期把资本投入到高回报机会中，并持续扩大竞争优势。

资本配置要看管理层如何在再投资、扩产、并购、分红、回购和现金储备之间做选择。

## 判断问题

1. 企业增长是否需要持续大额资本投入？
2. 新增资本的回报率是否仍高于资本成本？
3. 管理层扩张是纪律性投入，还是追逐规模？
4. 如果行业进入下行周期，公司是否能用成本优势反击？
5. 折旧、产能利用率和价格战如何影响长期复利？

## 台积电应用

台积电是重资本企业，因此“复利质量”要比轻资产消费企业更谨慎地判断。

支持复利的因素：

- 先进制程和先进封装仍有高需求。
- 规模、良率、客户绑定和工艺学习曲线带来领先优势。
- 《芯片浪潮》提到台积电折旧周期可能成为竞争武器：低谷建厂，高潮折旧，下一个低谷用已折旧产能参与价格竞争。

削弱复利的因素：

- 2026-2027 CAPEX 大幅上行，折旧压力可能吞噬部分利润。
- 海外建厂成本高于台湾本土，可能拉低整体回报率。
- 若 AI 需求或 CoWoS 扩产预期过高，新增资本回报率会下降。

## 本轮结论

台积电的资本配置不是简单的“扩产越多越好”。真正要跟踪的是：

```text
新增 CAPEX -> 先进制程/CoWoS 产能 -> 客户订单 -> 收入 -> 折旧后利润 -> ROIC
```

只有这条链条继续成立，台积电才符合巴芒意义上的长期复利企业。
