# 2026-W28 H1.2 存储周期更新

## 本周结论

H1.2 第一轮小循环已经跑通：原始资料进入 `03-raw`，沉淀为 source cards，回写到假设和概念页，并产出正式研究稿。

当前结论需要精确表述为：**AI 正在推动高端存储成长性，但没有消灭存储行业周期。HBM 是成长性最强的部分，服务器 DDR5 是二级传导，企业 SSD / AI SSD 是需要持续验证其成长性的新变量，消费存储目前仍然处于周期性而非成长性。**

## 分层判断

| 层级 | 产品 | 判断 |
| --- | --- | --- |
| 强成长化 | HBM3E / HBM4 / HBM4E | 被 AI GPU/ASIC、CoWoS、长约和认证共同锁定 |
| 结构性紧缺 | 服务器 DDR5 / SOCAMM | 受 AI 服务器需求和 HBM 挤压先进 DRAM 产能影响 |
| 成长化验证中 | 企业 SSD / QLC / AI SSD | AI 推理和高容量存储需求增强，但仍需验证订单持续性 |
| 周期仍强 | 消费 SSD / 普通 NAND / 低端 DRAM | 手机、PC 和消费电子需求仍是核心变量 |

## 下周跟踪

1. HBM4 / HBM4E：价格、认证和良率是否继续支持 HBM 成长性。
2. CoWoS booking 与 HBM 产能差：确认瓶颈是否仍在存储。
3. DDR5：服务器合约价是否继续上涨。
4. 企业 SSD / AI SSD：Google、Meta、Microsoft 等订单是否持续扩大。
5. 消费 SSD / 手机 / PC 库存：监控低端周期反证。
