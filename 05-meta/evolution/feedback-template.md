# 反馈模板（feedback-template）

> 当 AI 表现好或不好时，使用此模板记录反馈，帮助系统进化。

## 正面反馈

```markdown
- YYYY-MM-DD [类型：正面] [命令/技能] [具体表现] [为什么好] [建议保留/推广]
```

## 负面反馈

```markdown
- YYYY-MM-DD [类型：负面] [命令/技能] [具体问题] [为什么不好] [建议改进]
```

## 使用方式

1. 复制对应模板
2. 填写具体内容
3. 追加到 `05-meta/evolution/friction-log.md` 或单独反馈文件
4. 对 WorkBuddy 说「记录反馈」时，自动按此模板写入
