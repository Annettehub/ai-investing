## 核心逻辑

> 设备国产替代不是同时推进的，而是**接力式**的。当一道工序被攻克，瓶颈（binding constraint）会转移到下一道工序。这意味着：

1. **不同细分赛道的国内设备商在不同时间点获得验证机会** — 不是所有设备商同时受益
2. **投资节奏可预判** — 刻蚀先行(2026-2029) → 钨填/沉积(2027-2030) → 量测(2028-2031+)
3. **先发优势会衰减** — 某道工序攻克后，该赛道从"稀缺"变为"竞争"，利润可能下降
4. **最终瓶颈在量测** — "量测跟不上会同时拖慢所有工序的替代"

### 预期接力路线图

```
2026-2028  刻蚀(中微) ← 当前受益
    ↓ binding constraint 上移
2027-2029  钨填充(???) ← 国内未攻克, 最大瓶颈
    ↓
2028-2030  ONO共形镀膜(拓荆/微导) ← 高层数仍卡在Lam
    ↓
2028-2031+ 量测检测(中科飞测/精测) ← 最终瓶颈, 落后数代
```
