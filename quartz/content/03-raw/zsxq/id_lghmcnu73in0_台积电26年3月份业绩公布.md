台积电2026年3月业绩公布了，3月营收4151亿新台币，环比2月增30.7%，同比去年增45.2%。

环比增长如此之大的原因是2月份过年，工作日短，没啥其他原因，同比如此之高的原因是GB200/300的出货，一直维持高档，并塞满了产能。

台积电公布3月营收之后，1Q26的营收也就出来了1.13万亿新台币，依照美元兑新台币平均汇率，1Q26的美元营收在357~359亿美元之间，与笔者在1月10号早早在星球公布的模型357亿美元完全相符。

1月10号的模型如下图

![](images/6aaffd3fcadad9b51b7c.jpg)

1Q26的毛利率约在64.5%-65.8%之间，这是一个相当夸张的毛利率水平，但去年笔者已经充分预告过2026年以及以后的毛利率惊人。

之前笔者在星球的文章公布了多个从各环节去推导台积电营收的细部子模型，这里就不再过多说明。

未来台积电毛利率不断提高是我在2025年就重点提出的，如今都在一一落实。

台积毛利率的底层逻辑是，foundry的每一代制程价格永远是上涨的，折旧固定5年，所以每一个新节点折旧完毕，毛利率就会上一台阶，这从台积电30年的财报中能清楚发现这一条永远在上升的毛利率曲线。

这是foundry行业的逻辑，与之前我们讨论存储的周期性不同，但能保持毛利率永远上涨还有前提是稼动率必须至少恒定，不能往下跌。

意思就是只有稼动率维持不变，毛利率不断向上才能成立，如何保持稼动率不坠呢？那就是谁能引领摩尔定律，能引领摩尔定律的自然会有持续不断的订单已维持稼动率。

在2023年ChatGPT-4带火AI之前，foundry一直遵循如此的规律，不断稳步向上。如今AI时代来临，台积电到底为何能有更高的毛利率想上曲线呢？这不仅仅是因为产能应紧张生意更好，今天我们就来说明一下这个现象。

我们先看一下目前不同应用的芯片采用的制程情况。

我们以今年为例，大家能更清楚，今年下半年量产的最先进制程2nm由旗舰手机SoC(苹果A20，高通8gen6)以及旗舰CPU（AMD Zen6）采用。

3nm的次节点由AI旗舰芯片采用，如英伟达Rubin，谷歌TPU v7p，加上手机次旗舰（苹果A19，高通8gen5)，CPU的次旗舰AMD Zen5，高端矿机，旗舰显卡，HBM4 base die， 等

5/4nm为落后两代的次次节点，主要生产次旗舰的AI芯片，如英伟达blackwell与CSP ASIC，中低端手机Soc（苹果A18，联发科9400)，中低端CPU，高端Switch，次旗舰显卡，旗舰自动驾驶ADAS，旗舰智能座舱芯片，高端基站主控，HBM3e base die，AR/VR，旗舰FPGA等。

也就是说目前手机跟CPU采用最先进制程，AI采用次节点制程的产业态势，最大的应用分开不同节点，可以让台积电营收最大化，如果全部应用都挤在同一个节点，那台积电的收入增长将变得有限。

AI大火之后，需求节节攀升，虽然出货量级只有千万颗级别，与手机/CPU都是数亿级别的出货量相比低很多，但AI芯片面积不断地放大，从600mm2到850mm2，一个die不够用变成两个die，两个die不够明年还会有4 die ，也就是wafer的使用面积不断攀升，目前已接近CPU对wafer的需求量了，且还在高速增长中，很快将超越手机成为最大的应用。

AI的需求不断激增，除了最先进节点2nm给旗舰手机跟CPU用不影响以外，次节点3nm的产能无法满足AI以及其他应用的需求，所以发生产能外溢现象，台积电采用跨厂的多制程协同至关重要。

也就是说Rubin的关键层在台南Fab 18B的3nm工厂做，旁边5nm的Fab 18A可以生产非关键层，而台中7nm的Fab15B可以生产BEOL的金属层，如此一来2nm/3nm/5nm都将满产，7nm也会大大提高稼动率。

在AI时代之前，先进制程只有手机跟CPU两大应用，基本上只能填满最先进的两个节点，第三先进的节点稼动率开始下降，这种情况从智能手机开始便是如此，维持了接近20年，直到AI降临。

所以AI给台积电带来的是明面上的营收最大化的增长，最先进，次先进，次次先进全部爆满，甚至落后3代的7nm也大大提高的稼动率。

除了明面上的营收增长，台积电毛利率同样节节攀升，市场的理解是各节点稼动率提高，生意好，毛利率自然高，但这里面还叠加一个鲜为人知的逻辑。

刚才我们提到了Foundry的行业逻辑是只要稼动率能维持，毛利率将永远向上。但现在这个向上的毛利率将因为AI有更大幅度的上扬。

如上所述，AI时代之前，台积电只能维持两代制程满产，但是这两代最先进的工厂都是折旧中的低毛利工厂。

落后2~3代的工厂，比如现在的5nm跟7nm工厂已经折旧完毕，毛利率极高（70%+)，这些5年折旧完的最高毛利率Fab稼动率因为产业规律都是低的。

也就是这些折旧完的Fab用不高的稼动率，极高的毛利去滋养正在折计中的两座最先进制程工厂，为何是两座？因为摩尔定律两年到三年推进一代，台积电5年折旧期，所以大约都是维持两个世代的最新进节点正在折旧中。

比如2023年GhatGPT-4带动AI浪潮之前，当年台积电刚量产3nm，由于产能还在爬坡，且要给苹果赶货，所以稼动率必然是100%，此时3nm只有苹果一家，其他人的旗舰产品都挤在5/4nm，由于真正有量的苹果独自提升到3nm。

同时因为苹果旗舰从4nm转到3nm，5/4nm空出约60kwpm的大量产能缺口，所以5nm除了手机跟CPU的旗舰以外，还有大量补充了次旗舰产品，让5nm的稼动率也能保持满产。

但次旗舰产品从7/6nm提升到5nm，这导致7nm稼动率崩跌倒5成以下，因为7nm已是最低端的千元机产品，他没办法承接更成熟制程往前推进的芯片。

2023年状况是还在折旧中的3nm满产，还在折旧中的5/4nm满产，已折旧完毛利率超高的7nm稼动率低于50％。这是AI之前的常态，台积电7nm以上的十多座折旧完的工厂用较低的收入但极高的毛利率供养着前面两个5/3nm还未折旧完的最先进工厂。

如今AI造成的产能外溢，台积电非常牛的搞出了多敞多制程的产能协同，其实我一直搞不太清楚，台积电怎能做到跨厂的overlay对准的，又如何能保持高良率的，这一点令人匪夷所思，但事实证明他们做到了还做得很好，这将完全改变，未来先进制程在AI时代的产能建设逻辑。有空我再针对性地写一篇，目前市场还没有对这点有洞悉的。

因为AI的产能外溢，目前的局面是折旧中的2nm满产，3nm满产（2027开始陆续结束折旧），这两座工厂都在折旧中，5/4nm满产，7nm可能没满产，稼动率大幅度提高。

5/4nm这个高单价且折旧完成的Fab还能满产对台积电的贡献巨大，明年3nm的fab也开始陆续折旧完毕，还得满产，这又进一步提高台积电毛利率。

总而言之这些要点，我从去年就不断反覆告知星球里同学，有的能理解，有的始终无法理解，这取决于大家不同的认知。

已知的2026年台积电未来每一个季度的业绩基本都会超预期或者指引高位，所以每一个月发业绩前，每个季度财报前，都是布局的好时机，这从去年说到现在了。

根据我的模型，全年必然超公司指引close to 30的 yoy，我早在1月10号给出的业绩模型，已经证明1Q的业绩跟我预测的一模一样，全年35%+的yoy，我相信即便放到年底也只有正负2%以内的误差，H2到时候再根据最新情况做一个更新即可。

意思就是只有稼动率维持不变，毛利率不断向上才能成立，如何保持稼动率不坠呢？那就是谁能引领摩尔定律，能引领摩尔定律的自然会有持续不断的订单已维持稼动率。

在2023年ChatGPT-4带火AI之前，foundry一直遵循如此的规律，不断稳步向上。如今AI时代来临，台积电到底为何能有更高的毛利率想上曲线呢？这不仅仅是因为产能应紧张生意更好，今天我们就来说明一下这个现象。

我们先看一下目前不同应用的芯片采用的制程情况。

我们以今年为例，大家能更清楚，今年下半年量产的最先进制程2nm由旗舰手机SoC(苹果A20，高通8gen6)以及旗舰CPU（AMD Zen6）采用。

3nm的次节点由AI旗舰芯片采用，如英伟达Rubin，谷歌TPU v7p，加上手机次旗舰（苹果A19，高通8gen5)，CPU的次旗舰AMD Zen5，高端矿机，旗舰显卡，HBM4 base die， 等

5/4nm为落后两代的次次节点，主要生产次旗舰的AI芯片，如英伟达blackwell与CSP ASIC，中低端手机Soc（苹果A18，联发科9400)，中低端CPU，高端Switch，次旗舰显卡，旗舰自动驾驶ADAS，旗舰智能座舱芯片，高端基站主控，HBM3e base die，AR/VR，旗舰FPGA等。

也就是说目前手机跟CPU采用最先进制程，AI采用次节点制程的产业态势，最大的应用分开不同节点，可以让台积电营收最大化，如果全部应用都挤在同一个节点，那台积电的收入增长将变得有限。

AI大火之后，需求节节攀升，虽然出货量级只有千万颗级别，与手机/CPU都是数亿级别的出货量相比低很多，但AI芯片面积不断地放大，从600mm2到850mm2，一个die不够用变成两个die，两个die不够明年还会有4 die ，也就是wafer的使用面积不断攀升，目前已接近CPU对wafer的需求量了，且还在高速增长中，很快将超越手机成为最大的应用。

AI的需求不断激增，除了最先进节点2nm给旗舰手机跟CPU用不影响以外，次节点3nm的产能无法满足AI以及其他应用的需求，所以发生产能外溢现象，台积电采用跨厂的多制程协同至关重要。

也就是说Rubin的关键层在台南Fab 18B的3nm工厂做，旁边5nm的Fab 18A可以生产非关键层，而台中7nm的Fab15B可以生产BEOL的金属层，如此一来2nm/3nm/5nm都将满产，7nm也会大大提高稼动率。

在AI时代之前，先进制程只有手机跟CPU两大应用，基本上只能填满最先进的两个节点，第三先进的节点稼动率开始下降，这种情况从智能手机开始便是如此，维持了接近20年，直到AI降临。

所以AI给台积电带来的是明面上的营收最大化的增长，最先进，次先进，次次先进全部爆满，甚至落后3代的7nm也大大提高的稼动率。

除了明面上的营收增长，台积电毛利率同样节节攀升，市场的理解是各节点稼动率提高，生意好，毛利率自然高，但这里面还叠加一个鲜为人知的逻辑。

刚才我们提到了Foundry的行业逻辑是只要稼动率能维持，毛利率将永远向上。但现在这个向上的毛利率将因为AI有更大幅度的上扬。

如上所述，AI时代之前，台积电只能维持两代制程满产，但是这两代最先进的工厂都是折旧中的低毛利工厂。

落后2~3代的工厂，比如现在的5nm跟7nm工厂已经折旧完毕，毛利率极高（70%+)，这些5年折旧完的最高毛利率Fab稼动率因为产业规律都是低的。

也就是这些折旧完的Fab用不高的稼动率，极高的毛利去滋养正在折计中的两座最先进制程工厂，为何是两座？因为摩尔定律两年到三年推进一代，台积电5年折旧期，所以大约都是维持两个世代的最新进节点正在折旧中。

比如2023年GhatGPT-4带动AI浪潮之前，当年台积电刚量产3nm，由于产能还在爬坡，且要给苹果赶货，所以稼动率必然是100%，此时3nm只有苹果一家，其他人的旗舰产品都挤在5/4nm，由于真正有量的苹果独自提升到3nm。

同时因为苹果旗舰从4nm转到3nm，5/4nm空出约60kwpm的大量产能缺口，所以5nm除了手机跟CPU的旗舰以外，还有大量补充了次旗舰产品，让5nm的稼动率也能保持满产。

但次旗舰产品从7/6nm提升到5nm，这导致7nm稼动率崩跌倒5成以下，因为7nm已是最低端的千元机产品，他没办法承接更成熟制程往前推进的芯片。

2023年状况是还在折旧中的3nm满产，还在折旧中的5/4nm满产，已折旧完毛利率超高的7nm稼动率低于50％。这是AI之前的常态，台积电7nm以上的十多座折旧完的工厂用较低的收入但极高的毛利率供养着前面两个5/3nm还未折旧完的最先进工厂。

如今AI造成的产能外溢，台积电非常牛的搞出了多敞多制程的产能协同，其实我一直搞不太清楚，台积电怎能做到跨厂的overlay对准的，又如何能保持高良率的，这一点令人匪夷所思，但事实证明他们做到了还做得很好，这将完全改变，未来先进制程在AI时代的产能建设逻辑。有空我再针对性地写一篇，目前市场还没有对这点有洞悉的。

因为AI的产能外溢，目前的局面是折旧中的2nm满产，3nm满产（2027开始陆续结束折旧），这两座工厂都在折旧中，5/4nm满产，7nm可能没满产，稼动率大幅度提高。

5/4nm这个高单价且折旧完成的Fab还能满产对台积电的贡献巨大，明年3nm的fab也开始陆续折旧完毕，还得满产，这又进一步提高台积电毛利率。

总而言之这些要点，我从去年就不断反覆告知星球里同学，有的能理解，有的始终无法理解，这取决于大家不同的认知。

已知的2026年台积电未来每一个季度的业绩基本都会超预期或者指引高位，所以每一个月发业绩前，每个季度财报前，都是布局的好时机，这从去年说到现在了。

根据我的模型，全年必然超公司指引close to 30的 yoy，我早在1月10号给出的业绩模型，已经证明1Q的业绩跟我预测的一模一样，全年35%+的yoy，我相信即便放到年底也只有正负2%以内的误差，H2到时候再根据最新情况做一个更新即可。
