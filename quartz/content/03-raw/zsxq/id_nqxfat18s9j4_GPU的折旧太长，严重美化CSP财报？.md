# GPU的折旧太长，严重美化CSP财报？

最近看空AI是主流阿，Michael burry确实火了一把，昨天针对格隆汇那个引导性很强的但偏颇的报导，表达了对AI企业互相投资的一些看法。

今天咱们针对Michael burry这个推文。这推文我们可以简化为：

· 大前提：在严格财务准则下，不能创造显着经济利益的资产，其账面价值应被低估。

· 小前提：旧芯片（如A100）因能效低，其直接提供AI云服务的利润微薄。

· 结论：因此，这些旧芯片“本身也不值什么钱”。

**整篇推的问题就出在这个结论。 这是一个错误的推论。**

从“作为主力服务工具时不赚钱”（小前提），无法直接推出“作为资产整体不值钱”（结论），这中间缺失了一个关键环节，该资产是否还有其他能创造价值的用途？

他把旧芯片存在的“事实状态”冠上没有价值，推中引用旧芯片就像只在感恩节飞一下的旧飞机，其存在价值极其有限，近乎于零。

我想很多对『产业没概念』或缺乏缜密逻辑的人又会高呼，说得真好，那没效率的旧芯片就是如此，AI的水分太大了。

但事实上，这又是一个似是而非缺乏逻辑的引导性说法。

Burry引用了正确的会计概念，比如他清楚地指出 GAAP 关注的是「 economic benefits经济利益」，而不是单纯的「physical utilization物理使用 」。这点在会计上是无可辩驳的。

正是这一点让他收获许多没有产业知识懂财会的研究员的支持。但问题就出在他带入的产业事实是错误的，这必然会得出错误的结论。

他的错误在于将旧晶片的价值从「极度高效率」贬低为「没有效率」甚至「没有价值」。

一个AIDC可以利用已摊提成本的旧晶片来执行低利润但仍有收益的任务。

只要旧晶片提供的收益仍大于其电费和维护费，它就依然有正的经济利益。

**从产业层面我们得来审视一下，所谓旧的，能耗低下的GPU到底还有什么用途？**

1. 非生成式、低密度 inference

 · Recommendation 系统

 · 传统 NLP、BERT-base inference

 · 搜寻 ranking

 · fraud detection

 2. 批量、离线、非即时 workloads

 · log processing

 · embedding refresh

 · feature extraction

 · batch pretraining 小模型

 3. codec / video / transcoding / 媒体处理

 很多上好几代的A10、T4、甚至十年前的老 K80 都还在做这些工作。

除了上述作用，旧芯片推进资本支出与运营支出的平衡，也有很大作用。

对CSP来说，旧 GPU 降价后租金很低，只要不是每瓦性能极端敏感，就仍然是 ROI 更高的选择。

**更重要的是，未来 AIDC 会形成「分层算力市场」**

Tier 1：最新 GPU（Blackwell、H200、MI300）

 · LLM training

 · 大规模即时 inference

 · 创建新模型

特点：

 · 每瓦性能关键

 · 资本回收期短

Tier 2：上一代 GPU（H100 / A100）

 · LLM fine-tuning

 · RAG embedding

 · 中型模型 inference

A100 不会消失，只是价格重新定位。

Tier 3：旧 GPU（V100 / T4 / A10）

 · 传统 ML

 · 推荐系统

 · video workloads

 · offline batch jobs

 · edge / private DC overflow

所以上一代或者上好几代的GPU并不是没价值，而是换到适合的位置

Burry 的核心观点 在会计逻辑上是正确的，这让他收获很多认可，但他故意将旧 GPU 没性能就没价值，AIDC 会淘汰所有老晶片这明显是错误的，而且是天大的错误。

实际上，旧晶片不会退出，只是转向不同功能，AIDC也将形成分层算力市场，最终效率不是唯一价值，成本 / 任务需求 / ROI 才是。

所以目前CSP把GPU的折旧年限定在5-6年并没有太大问题，在笔者看来非常合理。当然有人说是当缩短到4-5年我也认为没问题。

但目前AI刚刚在发展初期，明明就是资本支出大量爆发的时刻，我想任何一家公司都不会也没理由把折旧缩短。

更不可能依照Burry这种不尊重产业规则，GPU必须2-3年折旧才符合财会准则，这种断章取义的极端说法。

**为什么 Burry 选择这种极端说法？**

作为一位著名的空头投资人，他的目的是透过推文向市场发出强烈的警告讯号，指出潜在的会计风险。

他必须使用极端的语言，如推文的「not worth much at all」、「confuses utilization with value creation」）来强调核心问题：大型科技公司正利用折旧年限来平滑和美化其惊人的资本支出压力。

他不是在写一份审计报告，而是在发布具备影响力的意见。

如果他只是说「折旧应该从 5 年改成 4 年」，那市场会忽略他的意见，因为5年改成4年问题不大，也不够耸动。

但他先指出了财会准则，在完全无视旧芯片的价值情况下，意有所指的指出GPU只有2-3年的价值，目前所有CSP都存在美化帐面的严重问题，这样一来，则能引起广泛关注和讨论。

重点是目前正个市场正在下行修正阶段，如此一来更能激发市场对AI泡沫的疑虑。

未来AI泡沫到底如何，或者会以什么形式发生，我在昨天的文章已经有阐述，过段时间我在写一篇更全面的，针对未来AI全面看法的文章。
