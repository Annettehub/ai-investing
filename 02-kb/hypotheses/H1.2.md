# H1.2 [D1-Trend] 存储周期特性是否向成长特性转变

## 核心命题

AI 正在推动高端存储成长性，但没有消灭存储行业周期。HBM 是成长性最强的部分，服务器 DDR5 是二级传导，企业 SSD / AI SSD 是需要持续验证其成长性的新变量，消费存储目前仍然处于周期性而非成长性。

## 当前判断

- 状态：active-tracking
- certainty：72%
- 正式研究稿：`04-output/reports/H1.2-storage-restructured-2026-07-12.md`
- 长期跟踪表：`04-output/tracking/H1.2-storage-tracking.md`

- 交叉验证来源：Morgan Stanley NAND 研报、三星存储/代工笔记、DRAM/NAND 深度分析、闪存专家访谈。

## 三层级 + 两机制框架

| 层级 | 产品 | 属性 | 核心驱动 |
|------|------|------|----------|
| 第一层 | HBM3E / HBM4 / HBM4E | 强成长性 | 直接绑定 AI GPU/ASIC 出货、CoWoS 配套、长约锁定 |
| 第二层 | 服务器 DDR5 / SOCAMM | 结构性紧缺 | HBM 挤占先进 DRAM 产能，AI 服务器 CPU 侧需求提升 |
| 第三层 | 企业 SSD / QLC / AI SSD | 成长性验证中 | AI 推理 KV cache、向量库、低延迟存储 |
| 反证区 | DDR4 / LPDDR / 消费 SSD / 普通 NAND | 周期仍强 | 手机/PC 需求弱，价格上涨更多来自供给扰动 |

**机制一：LTA（长期供货协议）**
锁量不必然锁价，但会缓冲周期波动，并制造存量/增量价格差异。

**机制二：产能竞争**
三大厂内部是零和分配：HBM 增加会挤占先进 DRAM 产能，DDR5 被动紧缺；NAND 则更多依赖控产和技术升级。

## 逻辑传导链

> 台积电 CoWoS 扩产（+55%） -> CoWoS 技术绑定 HBM 需求 -> HBM 需求跳升 -> 先进 DRAM 产能挤占（1c 制程零和分配） -> DDR5 供给被动收缩 -> DDR5 价格跟涨；NAND 因无扩产 + QLC 良率瓶颈，走“技术升级替代产能扩张”路径。

## 关键交叉验证

| 命题 | 主锚资料 | 其他资料 | 结论 |
|------|----------|----------|------|
| HBM 供给紧 | 2026 HBM 产能 +37.5%，但 CoWoS/HBM 反推显示仍有缺口 | DRAM 行业报告、三星访谈报告均支持 HBM/服务器 DRAM 紧 | 一致 |
| DDR5 被 HBM 挤压 | DDR5 价格 2026/2027 继续上行，驱动因素为产能向 HBM 倾斜 | 三星访谈报告支持先进 DRAM 分层 | 一致 |
| NAND 高端分化 | NAND 2026 产能 -3.2%，产值 +47.8%，企业 SSD 由 AI 拉动、消费需求一般 | Morgan Stanley 和闪存专家访谈支持 eSSD/AI SSD 强 | 一致 |
| 厂商控产 | DRAM 扩产保守，NAND 控产保价 | 多份资料提到 LTA、控产、技术升级替代大扩产 | 一致 |

## 保留分歧

| 分歧 | 说明 | 当前处理 |
|------|------|----------|
| 产能口径不同 | DRAM/HBM 为万片/月，NAND 为万片/年；其他资料也有 bit growth 口径 | 以吴梓豪产业研报为主锚，其他只做方向验证 |
| LTA 是锁量还是锁价 | 有资料强调锁供应和预付款，有资料强调年度重谈 | 视为降低周期波动，不视为永久锁价 |
| NAND 是否成长性 | Morgan Stanley 偏乐观，专家访谈显示 eSSD 也可能有客户砍单 | 只把高端企业 SSD 列为“成长性验证中” |
| HBM 价格涨幅 | DRAM 行业报告的 60-100% 与吴梓豪全年均价涨幅口径不同 | 不将 60-100% 用于收入测算，只作为定价能力参考 |

## 反证与 Kill Metrics

| 条件 | 阈值 / 触发 | 影响 |
|------|-------------|------|
| HBM 合约价连续回落 | 两个季度累计下跌超过 15% | 下调 HBM 成长性判断 |
| HBM4 良率快速突破 | 领先厂商良率突破 70%，且价格回落 | 供给缺口可能收窄 |
| 服务器 DDR5 合约价回落 | 环比下跌超过 15% | 说明紧缺可能只是短期错配 |
| LTA 未能落地或重谈 | 大客户长约覆盖率低于预期 | 周期波动重新增强 |
| AI eSSD 订单下修 | 主要 CSP 连续两个季度下修订单 | 下调 NAND 成长性判断 |
| 消费端库存上升 | 手机、PC、cSSD 库存持续升高 | 低端周期拖累整体 |
