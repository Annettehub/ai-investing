# S2.1 先进制造瓶颈、先进封装和国产替代

- **状态**: 活跃
- **旧编号**: H2.2 / H7.2 / H4.1 / H4.2 / H6.1 / H7.3
- **层级**: S 产业结构与价值捕获
- **风险标签**: geo, policy, capacity
- **certainty%**: 60%
- **投资方向**: 纯结构判断，不绑定仓位
- **最后更新**: 2026-07-21

## 核心命题

AI 硬件的价值捕获不只由芯片设计决定，而由**先进制程、先进封装、HBM/存储、设备材料、出口管制和国产替代路径**共同决定。

更具体地说：

1. **先进制造是全球 AI 硬件的硬约束**：台积电的先进制程、先进封装和系统集成能力决定 NVIDIA、AMD、ASIC、HBM 相关订单能否转化为交付。
2. **先进封装是 GPU 与 ASIC 订单的共同瓶颈**：CoWoS / SoW-X / 2.5D / 3D IC 不只是封装环节，而是系统性能、良率和交付节奏的核心约束。
3. **国产替代不是一条直线，而是瓶颈接力**：成熟制程、先进制程、存储、刻蚀、沉积、钨填、量测等环节会按约束强弱依次成为瓶颈。
4. **地缘政策改变价值链位置**：出口管制和供应链区域化会改变谁能拿到设备、产能、订单和利润。

## 合并后的子问题

| 子问题 | 原内容来源 | 当前判断重点 |
|---|---|---|
| 先进制程与封装双重壁垒 | 台积电壁垒判断 | 台积电先进制程、封装和客户生态构成系统壁垒 |
| 先进封装稀缺性与价值捕获 | CoWoS/先进封装判断 | 先进封装更适合由 Foundry 向下兼容，传统 OSAT 话语权有限 |
| 出口管制与成熟制程自主 | 出口管制判断 | 制裁切香肠式升级，国内成熟制程和部分设备获得自主窗口 |
| 半导体供应链区域化 | 区域化判断 | 供应链从效率优先转向安全和区域冗余，改变成本与利润分配 |
| 国产 AI 芯片需求与产能约束 | 国产 AI 芯片判断 | 国内 AI 芯片需求存在，但受先进制程、HBM、封装和软件生态约束 |
| 国产设备瓶颈接力 | 国产设备判断 | 某一道工序突破后，瓶颈会向下一道工序迁移 |

## 判断链条

```text
全球 AI CAPEX / 国产 AI 需求
-> 芯片设计订单
-> 先进制程产能
-> HBM 与先进封装
-> 设备材料与良率
-> 可交付芯片 / 模组 / 服务器
-> 收入、毛利和现金流分配
```

## 关键结构

| 环节 | 全球链条 | 国内链条 | 主要瓶颈 | 价值捕获观察点 |
|---|---|---|---|---|
| 先进制程 | TSMC / Samsung / Intel IFS | SMIC 等 | 良率、EUV/DUV、客户验证 | HPC 收入、先进节点产能、客户迁移 |
| 先进封装 | TSMC CoWoS / SoW-X，部分 OSAT | 长电等封测 + 国内设备材料 | interposer、良率、产线经验 | CoWoS 产能、封装 ASP、客户分配 |
| HBM/存储 | SK Hynix / Samsung / Micron | 长江存储、长鑫等 | 堆叠、良率、客户认证 | HBM 产能、ASP、DDR5/NAND 挤出效应 |
| 设备材料 | ASML / AMAT / Lam / TEL / KLA | 北方华创、中微、拓荆、中科飞测等 | 光刻、刻蚀、沉积、量测 | 订单、验证节点、国产替代率 |
| 区域化 | 美国、日本、台湾、韩国、东南亚 | 中国本土链条 | 政策、关税、制裁、交付安全 | 产能迁移、成本上升、区域溢价 |

## 复核清单

1. 台积电 HPC / AI 相关收入、毛利率和资本开支是否同步上行。
2. CoWoS 与 HBM 产能是否继续成为 GPU/ASIC 订单的分配约束。
3. 国内云厂商 CAPEX 有多少进入国产 ASIC、国内服务器企业、SMIC、国产封测、国产设备材料和国产存储。
4. 国产 AI 芯片采购是否能转化为真实生产负载，而不是库存、政策配套或低利用率部署。
5. 国产设备突破后，下一个瓶颈是否如预期迁移到钨填、沉积、量测或良率控制。
6. 供应链区域化是否带来成本上升，同时给本土设备、材料、封测或代工带来收入机会。
