  - 稼动率93.5%接近满产 [知识星球]
  - 中芯Q2财报大超预期 #59; 1Q26模型更新 #114; 战略转型分析 #42

## 第一章：产业链位置拆解

```
上游设备/材料
├─ ASML (DUV光刻机，EUV禁售)
├─ AMAT/Lam/TEL (刻蚀/沉积)
├─ JSR/信越/应化 (光刻胶，已断供报价)
├─ 国产设备商 (北方华创/中微/拓荆，部分替代)
    ↓
[晶圆代工] ——— 【中芯国际】
│   核心产品：7nm(N+2/N+3) → 14nm/12nm → 28nm/45nm+
│   在建工程：4个28nm新厂(京城/东方/深圳/天津西青)
    ↓
[芯片设计客户]
├─ 华为海思 (麒麟/昇腾/鲲鹏) — 7nm绝对主力客户
├─ 寒武纪 (AI芯片)
├─ 韦尔/格科 (CIS)
├─ 海光/飞腾/龙芯 (CPU)
    ↓
终端客户：智能手机 / AI数据中心 / 汽车 / IoT
```

```

                       ↓
              中芯7nm产能从20k→30k
```
