# Serenity 投研方法论

## 一句话本质
不是找"便宜公司"，而是找"市场还没连起来的结构性卡点"。

## 七层框架（F-A-B-E-S-R-P）

| 层级 | 英文 | 核心问题 | 小苔花映射 |
|-----|------|---------|-----------|
| F | Funnel | AI capex 正在流向哪里？ | `/idea` 信号收集 |
| A | Architecture | 当前技术路线在哪一代？下一代卡点？ | `concepts/` 技术路线 |
| B | Bottleneck | 哪个环节最卡、最难扩产、最容易提价？ | `walls/` 风险+预警 |
| E | Exposure | 哪家公司对这个瓶颈暴露最纯？ | `entities/` 公司档案 |
| S | Security Structure | 股东回报会被债务/稀释/权证吞掉吗？ | `hypotheses/` 假设验证 |
| R | Repricing | 市场何时会发现？催化是什么？ | `/today` 每日检查 |
| P | Portfolio | 用核心仓/篮子/事件仓/期权表达？ | `/trade` 决策矩阵 |

## 四个反直觉点

1. **好瓶颈 ≠ 好股票**（必须经过证券结构过滤）
2. **当前代际赢家 ≠ 下一代赢家**（必须判断架构代际）
3. **下游赚大钱 ≠ 上游赚小钱**（价值链错位）
4. **信息综合 ≠ 独家信息**（更快把公开信息拼成可交易结构）

## 评分表（100分制）

| 模块 | 分值 | 问题 |
|-----|------|------|
| 资本流确定性 | 12 | 下游 capex 是否已明确？ |
| 瓶颈强度 | 18 | 供给是否真卡？难扩产？ |
| 架构位置 | 12 | 当前/过渡/下一代核心？ |
| 暴露纯度 | 13 | 公司收入能否吃到瓶颈？ |
| 证券结构 | 15 | 债务/稀释/权证/锁定期？ |
| 重估催化 | 12 | 未来3-12个月是否有催化？ |
| 风险可控 | 10 | 融资/客户集中/替代风险？ |
| 仓位适配 | 8 | 适合core/basket/event/options？ |

| 总分 | 动作 |
|---:|------|
| 80-100 | 核心候选，继续深挖 |
| 65-79 | 篮子或中等仓位，等催化 |
| 50-64 | 观察清单，小仓位 |
| 0-49 | 暂不交易 |
