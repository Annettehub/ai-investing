## 维护原则

- 单层能解释的主题，归入对应层级；跨层页只保留多层联动问题。
- 每个跨层主题必须能回答“哪个变量改变会穿透多个层级”。
- 主文档只保留一份，跨层页做索引和触发器，不复制全文。

## 分析模板

每个跨层主题后续更新时，按以下格式追加：

| 字段 | 说明 |
| --- | --- |
| 触发事件 | 哪个新闻、纪要、数据或技术进展触发 |
| 影响层级 | L1/L2/L3/L4/L5 哪几层受影响 |
| 一阶影响 | 直接改变什么变量 |
| 二阶影响 | 可能改变哪条投资假设或公司利润分配 |
| 需要回看 | 需要更新哪些 concept/entity/hypothesis |
| 结论状态 | 观察 / 待验证 / 已确认 / 已证伪 |
