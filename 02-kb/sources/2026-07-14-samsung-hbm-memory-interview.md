# 三星 HBM 与存储市场展望：专家访谈纪要

## 一句话

这份三星专家访谈支持服务器 DRAM 长协锁量与 HBM4 技术升级，但关键数字尚缺原始合同、公司公告或第二个独立来源，适合作为跟踪信号，不宜直接作为最高置信度测算锚点。

## 关键提炼

| 主题 | 信息 | 当前处理 |
| --- | --- | --- |
| 三星 DRAM 产能 | 约 60 万片/月；平泽四厂 2027 年起释放，设计产能 20 万片/月 | 与既有产能框架方向一致，口径待核 |
| 服务器 DRAM 长协 | 三星约 70% 服务器 DRAM 产能已锁定 | 支持 LTA 钝化周期，待公司或客户侧交叉验证 |
| 美光预付款 | 约 220 亿美元 | 金额较大，列为高优先级待验证项 |
| HBM4 | 三星采用 1c DRAM + 12nm base die | 支持 HBM 竞争转向 DRAM 制程与 base die 协同 |
| HBM4E | 访谈判断美光采用台积电 3nm base die 可能更具优势 | 属专家判断，不视为已确认结果 |
| 价格 | 2026 年每季度环比上涨 10%-15%，2027 年涨幅放缓 | 支持景气延续，也提示增长斜率见顶风险 |
| 中国供给 | 长鑫 HBM 约处于 HBM2 阶段，落后主流 3-4 代 | 支持国内高端 HBM 仍受技术约束 |

## 对 H1.2 的影响

- **支持**：服务器 DRAM 长协覆盖和 HBM4 技术升级提高高端存储收入可见度。
- **中性**：2027 年价格涨幅放缓，说明成长性增强不等于价格永远高增长。
- **待验证**：70% 锁定比例、220 亿美元预付款，以及 AI 服务器存储成本占比等数字不能在未交叉验证前进入正式测算。
