### 3. 国产5nm三连问
- **没有EUV，华为5nm芯片哪来的？** → DUVi + 多重曝光
- **没有EUV，即将上市的国产5nm从哪来？** → N+3工艺，Mate 70搭载
- **没有EUV，国产5nm如何生产？** → 技术细节全解（PSM/OPC/SADP/DSA/ILT）
- **核心结论**：能用，但成本极高，良率低，产能有限

### 4. 麒麟芯片出货展望及国产先进制程产能测算

- 升腾910B/C排产优先于麒麟（AI > 手机）
- 花系手机策略转向"重质不重量"

## 对00981假设的影响

| 假设 | 影响 |
|------|------|
| H4 7nm良率爬坡 | 麒麟/升腾共享产能 → 中芯先进制程产能分配是零和博弈 |
| H3 断供硬化 | 国产先进制程需求远超供给 → 成熟制程持续受益 |
| H6 叙事溢价 | ⚠️ 寒武纪估值模式可能复制到中芯（高预期 vs 低实际AI贡献） |
