# SRC-ZSXQ-Sanctions — 制裁/关税/地缘政治全景

> 来源：知识星球「半导体大佬的会议室」(吴梓豪leslie) | ~15篇 | 截止2026.6.25
> 包含：BIS制裁演进、MATCH Act、实体清单、稀土博弈、原产地规则、特朗普关税

## 核心论点

### 1. 美国制裁"切香肠"式演进（完整时间线）
- **2017**：中兴伊朗禁运认罪
- **2019**：华为实体清单
- **2020.09**：要求所有供应商断供华为（台积电三星停供）
- **2020底**：中芯国际实体清单（禁止10nm以下设备）
- **2022.10.07**：BIS 1007法案 → **全面对抗起点**（禁止14nm FinFET / 18nm DRAM / 128层 NAND）
- **2023.10**：BIS 1017新规（纳入1980i审批权 + H800禁售）
- **2024.09**：GAA + 1970i/1980i双重封锁
- **后续**：MATCH Act（打向中国半导体的真正命门）、200家企业可能进入实体清单

### 2. 华为应对策略（多点开花）
- 2020年断供前就开始谋划自制芯片
- 入主福建晋华（DRAM）→ 中芯宁波（射频/PMIC）
- 透过深重投100%持股新凯莱做零部件开发
- 二手设备商搜刮全球14nm设备 → 松山湖组建10k产能（2022下半年量产基站主控）
- 主导青岛芯恩、ICRD、PXW、PXX、SWX等多家Fab

### 3. 特朗普关税大战（2025.04更新）
- 美国对中国关税加征50% → 累计**125%**
- 中国反制加征84% → 双方互征**125%+**
- **等同于中美全面脱钩**：没有任何商品可承受100%+关税
- 在途货物豁免规则已明确
- 半导体原产地规则极其复杂（跨国制造 + 技术分类 + 三方政策）

### 4. 中国芯片出口真相（反常识）
- 2024年中国芯片出口$1595亿（历史次高），但**70%是外资在华生产**
- 出口目的地：台湾省(25%)、韩国(20%)、越南(15%)、马来西亚(12%)
- **中国进口全球7成芯片但只消费20-30%**：其余加工组装后外销
- 国产芯片以内销为主（含被迫国产化的实体清单企业）

### 5. MATCH Act —— 真正命门
- 打向中国半导体的更深层次限制
- 具体条款和影响范围（详见原文）

### 6. 稀土牌
- 中国限制稀土出口 → 全球半导体影响几何？
- 中美贸易谈判底定后的稀土博弈
- 实际威慑力有限（替代来源正在建立）

## 对00981假设的影响

| 假设 | 影响 | 方向 |
|------|------|------|
| W2 设备断供 | ✅ 核心输入 | 制裁只会加码不会放松（切香肠模式已被反复验证） |
| W3 材料断供 | ⚠️ 需关注 | 光刻胶（日本断供风险）、稀土反制效果有限 |
| W5 EDA/IP断供 | ⚠️ 间接影响 | MATCH Act可能扩展到EDA工具链 |
| H3 断供硬化 | ✅ 强支撑 | 每轮制裁都在修补漏洞（1007→1017→GAA+光刻机→MATCH） |
| H6 叙事 | 🔴 地缘风险 | 125%关税=事实脱钩，任何"全球化复苏"叙事都站不住 |
