# 三星存储与晶圆代工业务分析笔记

## 一句话

这份笔记提供了三星和行业产能的微观补充：高端 DRAM/HBM 依赖 EUV 与先进制程，CSP 长约正在锁定服务器 DRAM/HBM，低端 DRAM 和高端 DRAM 的分层会继续扩大。

## 关键提炼

| 主题 | 内容 |
| --- | --- |
| 全球 DRAM 产能 | 约 186 万片/月，三星、海力士、美光、长鑫分别约 66/50/50/20 万片/月。 |
| 全球 HBM 产能 | 约 32.6 万片/月，三星 14 万、海力士 15 万、美光 3.6 万。 |
| 2026 扩产 | DRAM 增量主要服务服务器；HBM 预计继续扩张，但仍受良率、认证和封装限制。 |
| LTA | 大客户长约锁供应和价格框架，预付款约束增强，服务器 DRAM/HBM 的周期波动被钝化。 |
| 分层 | 如果低端 1a/D1Z 供给宽松，高端 1b/1c/HBM 价格也未必同步下跌。 |
| NAND | 三星 2026-2029 更偏技术升级而非大规模新扩产。 |

## 对 H1.2 的影响

- 支持“高端存储和低端存储分层定价”的判断。
- 支持“HBM/服务器 DDR5 的供需结构已经不同于传统消费电子存储周期”。
- 需要注意：部分产能口径与吴梓豪 PPT 不完全一致，应以 PPT 为主锚，笔记作为微观补充。
