# 2026Q2 存储市场更新：DRAM 与 NAND 价格分化

## 一句话

价格上涨并未同步传导到所有产品：原厂和高端产品议价更强，消费 SSD 已出现库存与价格倒挂，进一步证明不能用一个存储价格指数判断全行业成长性。

## 关键提炼

| 信号 | 数据/描述 | 对 H1.2 的含义 |
| --- | --- | --- |
| DRAM 涨价 | Q1 约 80%-90%，Q2 预计超过 50%，Q3 涨幅预计降至 Q2 一半 | 景气仍强，但涨价斜率开始回落 |
| NAND 分层 | UFS +70%、wafer +25%、SSD +15% | 成本向下游传导不完整，产品间分化明显 |
| Mobile DRAM | 约 +80%，主要由低功耗 DDR4 停产驱动 | 更像供给收缩型周期，不是纯需求成长 |
| 服务器 DRAM | 64GB RDIMM 在 2,300 元遇阻后回到 2,150 元 | 提示价格存在接受度上限 |
| 消费 SSD | 库存积压、价格倒挂 | 构成“全行业成长化”的直接反证 |
| 原厂策略 | 额度销售、NCNR、预付款和长约逐步加强 | 提高供给纪律和价格控制力 |

## 对 H1.2 的影响

- **支持**高端/服务器存储与消费存储分层。
- **反证**“价格上涨等于需求成长”：部分涨价来自停产、控产和渠道策略。
- 与 Morgan Stanley 对消费产品涨价遇到天花板、服务器产品仍紧的判断方向一致。
