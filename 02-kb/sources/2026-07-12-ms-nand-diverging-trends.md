# Morgan Stanley：NAND Industry Outlook - Diverging Trends

## 一句话

这份署名卖方研报支持一个重要修正：NAND 不能再简单归为“弱周期品”，AI 推理和企业 SSD 正在把高端 NAND 拉出消费电子周期，但消费 SSD 仍然偏弱。

## 关键观点

| 主题 | 提炼 |
| --- | --- |
| AI 对 NAND 的拉动 | AI 推理工作负载提升企业 SSD、AI SSD 和低延迟存储需求，NAND 短缺可能延续到 2027。 |
| LTA | 长约提高供应可见度，也说明市场处于紧张状态；但 LTA 不是绝对价格锁定，需要结合年度议价。 |
| 高端企业 SSD | Kioxia、Sandisk、Micron、Phison、Silicon Motion 等均受益于高端 eSSD / 控制器需求。 |
| 主要风险 | 最大风险不是普通消费需求，而是 AI capex 放缓。 |

## 对 H1.2 的影响

- 强化“高端 NAND / 企业 SSD 成长化”的证据。
- 但也提示 NAND 内部必须拆分：企业级、AI SSD、QLC 与消费 SSD 的周期不同。
- 可作为吴梓豪 PPT 中“企业 SSD 需求由 AI 拉动”的外部交叉验证。
