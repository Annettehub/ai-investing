## 核心观点

1. **持续学习是最重要的投资能力**
   > "投资这项游戏的名字叫不停地学习。即便你接受过良好的训练，即使你天赋异禀，依然需要不断地学习。站在巨人的肩膀上，巴菲特比格雷厄姆好得多，总有一天会有人比我们做得更好。"
   - 关联 **小苔花系统设计**：闭环学习（read → think → write → review）是巴菲特方法的系统化。

2. **好生意不需要频繁交易**
   > "如果你所处的行业是你所理解的，并且你认为这是一个非常出色的行业，那么你就应该坚定持有它，根本不用担心。如果股价下降 25-30%，如果你有更多的钱，那就买更多。"
   - 关联 **00981**：如果确认 SMIC 是好生意（H1/H4/H6 综合判断），短期波动不应触发交易。

3. **品牌护城河的价值**
   > "喜诗糖果的案例 — 这是我们第一次购买品牌企业。对我们这些习惯于以 50 美分买 1 美元的人来说，这是一次艰难的跳跃。"
   - 关联 **H6**：00981 的"品牌"是中国半导体自主可控叙事，这是否构成类似品牌护城河？

4. **全球分销基础设施 = 护城河**
   > "当人们需要你的产品的时候，不管他们身处何方，你都必须得把产品送到那里。可口可乐二战时的全球工厂建设。"
   - 关联 **W5**（28nm 价格战）：SMIC 的客户网络和交付能力是否构成防御性资产？

5. **通过长时间做大量小事积累价值**
