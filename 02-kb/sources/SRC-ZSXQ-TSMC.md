### 2. CoWoS产能真相
- **2023年**：台积电47k + Amkor second source 16k = 全年63k → NV出货220-230万颗GPU
- **NV booking操作**：2023年初预订30k → 2月+5k → 4月+10k → 全年仍不够
- **2025年**：NV CoWoS booking约400k（卖方450k→390k的"砍单"是拍脑袋下调回归正常）
- **预付款博弈**：产能紧张时提前9-10个月付预付款；充足时只提前半年
- **台积电恐怖平衡**：CC魏每次法说会说CoWoS紧张 → 迫使客户早给预付款

### 3. 英伟达 vs 博通：GPU vs ASIC世纪对决
- 作者2023-2024建议路径：**All in NV（200→1255）→ 清仓NV转All in TSMC**
- 2025判断：NV挑战$2000亿营收（+70%）确定无疑，但**2026增长率将降至20%**
- NV市值路径：最高碰$4万亿 → 回落 → $2.5-3万亿强支撑
- **博通ASIC威胁**：百亿大单落地，未来三年ASIC增速可能超过GPU

### 4. 从CoWoS到SoW-X：系统级重构

- OIP推3D IC设计新标准
- CPO/OIO：硅光终局推演（SerDes 224G已近物理极限）
- SiC碳化硅做CoWoS？开玩笑
