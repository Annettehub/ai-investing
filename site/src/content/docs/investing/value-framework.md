# 价值投资框架

价值投资部分适合作为 AI 投研之外的底层方法论：风险、周期、逆向、钟摆理论和“无法预测但可以准备”。

## 当前网页整理方向

| 主题 | 对应问题 |
| --- | --- |
| 风险的本质 | 风险不是波动，而是永久损失和错误定价。 |
| 钟摆理论 | 市场情绪在乐观和悲观之间摆动。 |
| 逆向投资 | 共识极端时，反向判断才可能有超额收益。 |
| 准备而非预测 | 不把精力放在预测点位，而是构建情景和应对。 |
