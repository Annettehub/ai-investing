## 当前定位

这个网站是知识库的公开阅读层。仓库里的 `02-kb`、`03-raw`、`04-output` 仍然是主资料库，网站只挑选已经整理、适合公开阅读的内容。

## 阅读路径

| 模块 | 适合解决的问题 |
| --- | --- |
| 知识地图 | 先了解整个 AI 投研框架如何分层。 |
| AI 投研 | 阅读 GPU、ASIC、CPO、服务器供应链、资本开支等核心主题。 |
| 投资框架 | 沉淀价值投资、周期、风险和逆向思维。 |
| 内容管线 | 了解原始资料如何进入 `03-raw`，再沉淀到知识库。 |

## 维护原则

原始资料先进入 `03-raw`，经过整理后进入 `02-kb` 或 `04-output`。网站导航只放短标题和清晰主题，避免把长文件名直接暴露给读者。
