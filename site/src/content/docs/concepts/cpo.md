# CPO 光互联

CPO 是 AI 数据中心互联从可插拔光模块走向芯片级光互联的重要路线。它的核心驱动力是功耗、带宽密度和 SerDes 物理极限。

## 核心逻辑

| 问题 | 观察 |
| --- | --- |
| 为什么重要 | AI 集群规模扩大后，网络和互联功耗成为系统瓶颈。 |
| 技术路线 | 可插拔光模块、1.6T、CPO、OIO 逐步演进。 |
| 产业链 | 交换芯片、硅光平台、激光器、封装和整机系统需要协同。 |
| 投资映射 | 短期看 1.6T 光模块，长期看硅光和封装平台。 |

## 跟踪问题

- 1.6T 光模块放量节奏是否兑现。
- CPO 的可维护性和标准化问题能否解决。
- 台积电、Broadcom、NVIDIA 等平台型玩家如何布局硅光生态。
- 传统光模块公司在 CPO 时代的价值链位置是否被压缩。
