# 原始资料库

`03-raw` 是知识库的入口层。它保存尚未充分整理的原始资料，后续再沉淀到 `02-kb` 或 `04-output`。

## 当前来源

| 目录 | 类型 | 发布策略 |
| --- | --- | --- |
| `03-raw/podwise` | 播客转写、摘要、双语资料 | 先保留原文，适合整理后再公开。 |
| `03-raw/feishu` | 飞书文档同步 | 需要检查隐私和格式后再发布。 |
| `03-raw/weread` | 微信读书划线和想法 | 默认不整批公开，只抽取为读书笔记。 |
| `03-raw/wechat` | 微信来源资料 | 需要逐篇清洗。 |
| `03-raw/zsxq` | 知识星球资料 | 默认不公开原文，只沉淀观点和索引。 |

## 推荐流程

1. 新资料进入 `03-raw`。
2. 给资料补充来源、日期、主题和关键词。
3. 抽取概念、公司、假设和数据点。
4. 沉淀到 `02-kb` 或 `04-output`。
5. 适合公开的内容再进入 `site/src/content/docs`。
