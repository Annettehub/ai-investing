# 知识地图

当前知识库适合用“五层 + 跨层方法论”的方式展示。网页导航需要短标题，内部文件可以继续保留更长、更精确的研究命名。

| 层级 | 网页短标题 | 主要问题 |
| --- | --- | --- |
| L1 | 能源 | 数据中心是否有电、能否接入、成本如何变化。 |
| L2 | 芯片 | GPU、ASIC、HBM、先进制程、先进封装如何竞争。 |
| L3 | 基础设施 | 服务器、网络、光模块、CPO、液冷和数据中心建设。 |
| L4 | 模型 | Transformer、MoE、推理优化、开源与闭源模型生态。 |
| L5 | 应用 | Agent、Copilot、机器人、自动驾驶和商业化路径。 |
| Cross | 方法论 | 算力、能源、资本开支、供需周期和投资映射。 |

## 推荐网页结构

先把网站导航控制在 6-8 个一级主题以内。每篇页面用短标题展示，正文里再保留来源、编号、日期和详细证据。
