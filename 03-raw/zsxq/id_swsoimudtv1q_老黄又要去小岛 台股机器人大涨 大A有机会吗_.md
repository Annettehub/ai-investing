# 老黄又要去小岛 台股机器人大涨 大A有机会吗?

                半导体大佬的会议室](https://wx.zsxq.com/group/88888812815442)
        
        
            
                

![](images/39ac34f1b548792d9368.jpg)

                吴梓豪leslie
            
            2024年08月19日 13:56
        
        
            

![](https://article-images.zsxq.com/FvhWAI7kFiD5vvGomJYgEQr1wG33)

台北际自动化工业大展将在21日至24日登场，NV执行长黄仁勋将再度来台。外媒wccftech报导指出，机器人领域未来只会蓬勃发展，已经反映在台股走势上，带动包括所罗门在内的台股相关企业大幅上涨。

报导提到，NV已经将台湾视为业务不可分割的一部分，持续与台湾及企业活动保持密切联系，自动化市场的估值已经冲上50兆美元，围绕在NV人型机器人「Project GR00T」的炒作也相当盛行。

黄仁勋将在本周访台参加自动化工业大展，预料将再掀起一阵旋风，相关概念股的表现引发关注，所罗门近期股价大涨，今（19日）早盘也一度上涨超过3%，圆刚直冲涨停价49.55元，慧友一度锁在涨停价90元。

另外，广运、弘忆股、德律及新汉等概念股也都飙出超过6%的涨幅，成为台股上涨的动力之一。黄仁勋此次来台是否有新的发言备受关注，期待能藉此一窥产业趋势与脉动。

除了机器人题材之外，市场相当关注NV 28日公布的财报表现，被视为推升股市成长的重要观察指标，许多华尔街分析师纷纷上调预测，看好营收有望达到年增58%至286亿美元。
