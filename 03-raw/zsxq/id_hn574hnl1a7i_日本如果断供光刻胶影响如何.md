这事在高市早苗荒唐言论出口之时就可以预见端倪，事态演变下去光刻胶断供可能是大概率。

这也是小日子惯用手法，2019年对付韩国就是来这招。

光刻胶一直是笔者与行业内担心的事，它太重要了，国产化率又非常低，一但断供那对国内先进制程是毁灭的打击，但很可惜還是有很多人包含業內存在僥倖心理。

要明白这事有多严重，首先还是得从光刻胶本身谈起

光刻胶有非常多种，PCB，面板，LED，先进封装都要用到，但这些领域的光刻胶要求不高，主要还是线宽要求不高。

半导体用光刻胶是最高级别，分类主要以曝光的光源波长也可以说光刻机来分

1. G-line (436 nm)、i-line (365 nm)：对应0.25um以上的工艺。

2. KrF (248 nm)：对应0.13~0.25um工艺。

3. ArF (193 nm)

    ArF dry (干式) : 0.11um~65nm工艺

    ArF immersion（浸没式）：45nm~7nm

4. EUV (13.5 nm)：7nm~2nm

从曝光波长也就是光刻机种类来区分，针对不同的波长有不同的光刻胶，这属于光刻胶的大分类。

事实上不论哪种波长的光刻胶，都不是单一种光刻胶，而是每种波长的光刻胶高达数十种甚至上百种。

每种波长光刻胶之下，我们还能依照不同功能来区分

比如正/负胶，line/space 以及 contact-hole 专用配方，topcoat / BARC（底抗反射涂层），显影剂/去胶剂、以及不同硬掩膜（hard mask）材料 等。每一类都有专门配方。

所以光刻胶不是「一个产品」，即便是同一波长，比如A胶他也会有多个用途/多种化学配方/多个 tone 与不同解析度-灵敏度 trade-off 取舍。

**光刻胶用在哪？**

一颗先进制程芯片有大几十层，大致可分为器件层，金属层，介质层，接触孔层，钝化层等。

把这些功能层加工出来，主要就是透过反覆的增材与减材的逐层加工，一层一层把芯片所有功能层做出来，整个过程可能会超过上百层工序。

我们以国内用DUVi光刻机＋SAQP四重曝光制作7nm的工艺为例，由于多重曝光会显着增加 litho-step、mask 、以及所需的材料/处理步骤，最终整个加工步骤在120～130层左右。

其中需要光罩做图型定义的mask layer超40层，硬掩膜hard mask约20层，有些非关键层可以直接用光刻胶做软掩膜，剩下有一半是不需要光刻的沉积层，离子注入层以及平坦化层等等。

也就是说四重曝光的7nm芯片制程，需要用到光刻胶的超60层左右。

这代表制作一颗7nm芯片需要60种光刻胶？

事实上数量不是这么算的，60层有很多是同一个SKU，而一个layer也不一定只用一种光刻胶。

大类来说，这个7nm芯片制作过程需要用到 I胶，K胶，A胶以及浸没式A胶，因为7nm芯片，在高层数金属层也会用到i-line以及KrF这类成熟制程的光刻机。

单单针对ArFi的光刻胶，就多达数十种，vendor必须推出多个不同配方以满足不同的「解析度／灵敏度／线宽/孔径/溶解特性/耐刻蚀性/热稳定性／相容底层」的需求。

比如，不同的正胶 L/S（line/space）配方、接触孔（contact hole）优化配方、极高解析（低 LER）配方、低溶解度配方、低残留配方、双重图形化/间接 patterning 专用配方、以及配合某些硬掩膜的专用配方等。

每一种配方在化学组成，聚合物主体、PAG、溶剂、溶解抑制剂、添加剂等都会不同。

目前全球主要光刻胶供应商，JSR、TOK、信越等对于 ArFi 都有数十款SKU以供选择（按层、按用途、按曝光工具/数据窗）。

而且上述每一家都有自己擅长的配方，比如否种功能用某家的良率会更好，整体来说目前ArFi光刻胶，信越的评价最好，良率最高。

所有的ArFi胶有数十种，那制作一颗7nm芯片需要用到多少种呢？

我们以M1/M2这样的key metal layer用多曝制程来说，这会牵涉到不同的材料/涂层步骤，每一步都可能用不同配方，

首先，第一道光刻（mandrel）用 ArFi 正胶（line/space 配方) ;

刻蚀／形成 spacer，这牵涉到hard mask材料，而hard mask可能用不同的沉积材料/蚀刻配方。

接下来用氧化或氮化层沉积，再做 spacer，然后在用stripper去除 mandrel；

然后再做第二次光刻，如果是LELE，可能要另一种光刻胶或相同配方但不同曝光条件；

上述的一个多重曝光工艺要用到许多的不同功能的光刻胶，但还远不止与此，因为每一次coating涂胶前还得先涂一层BARC底部抗反射涂层。

浸没式光刻制程，coating之后，前烘完还得在光刻胶上涂一层topcoat用来保护光刻胶跟纯水直接接触。

也就是说光刻胶涂胶之前跟之后分别还要再上一层特殊功能的胶。

这也还没完，每次曝光后的developer还得用显影剂，刻蚀完之后还得用stripper去除残留光刻胶还有清洗液。

上述就是一个多重曝光需要用到光刻胶以及各种化学药剂的过程，而上述的所有化学品，在14nm以下的先进制程中，所有品类目前的国产化率都非常低。

即便门槛较低的显影液或去胶液整体国产化率还不错，但是针对先进制程的国产化率却不到20%，ArFi光刻胶，BARC以及topcoat则更低都在10%以下。

这些正是国产7nm芯片生产过程必须用到的化学品，没有这些化学品，刚刚公布的麒麟9030将没办法生产。

更麻烦的是，一个做先进制程的fab，其前段与后段、所有层，会用到上百种不同的 resists / BARC / topcoat / stripper / developer 。

这也是为什么「替换光刻胶供应商」不是换一桶胶就能完成的原因，因为每个 layer 的相容性、曝光窗口process window、热处理、黏着性、残留物、与光刻机与涂胶显影设备高度绑定。

也就是说用ASML的scanner配套TEL的track，每一个 layer 用的 resists / BARC / topcoat / stripper / developer也是必须是固定vendor与配方，研发能力强的fab甚至会要求vendor出独特的配方，配套自家工艺。

如果今天我们把ASML的scanner换成nikon或者国产光刻机推出，那上述的所有化学品都要有跟着全换一遍相配套的配方。

简单说就是未來如果设备换了，光刻胶与化学品也得跟着换相应的配方。

所以一直有很多人关心国产光刻机进度，却没有人关心光刻制程中还有一大堆要配套的设备与化学品神智还有计算光刻等软件，这是一整套的系统级工艺，每一项缺一不可，如此多环节配套且高度系统化的光刻工艺，确实大大增加了攻关难度。

目前全球光刻胶的主力供应商为 JSR、TOK、信越、Fujifilm、DuPont、Merck、Sumitomo等大型化学厂，这些公司在配方、纯度控制、批次稳定性、以及长期与 fab合作的认证生态方面非常成熟。

其中JSR、TOK、信越为D3，几乎垄断全球高端光刻胶市场，D3的ArFi市占约92%，EUV光刻胶更是达95%。

D3对高端光刻胶的绝对垄断，而且全部来自日本，这正是小日子有恃无恐的底气。

2019年小日子产经省宣布对韩国的光刻胶，高纯度氢氟酸，氟聚酰亚胺实施强力管控。

起因是韩国在对日强硬派总统文在寅上台后，韩国法院判决日本三菱重工与新日铁须向二战强征的韩国劳工赔偿，而日本以1965年的『日韩请求权协定』已完全解决所有历史赔偿问题，认为韩国违反国际法。

遂于2019年7月宣布光刻胶在内的三种化学品对韩出口管制。

日本管制的同时，2019年Q2～Q4三星因为光刻胶储备不足，营收同比下滑3成以上，利润暴跌53%。

而业务多元化不如三星的海力士则更为惨烈，营收接近腰斩，利润暴跌89%。

2019年韩国半导体出口同比下滑27%，一个国家的最大支柱产业出口大跌，拖累当年韩国GDP增长接近1个百分点。

韩国半导体行业被小日子狠狠地上了一课。

2019年11月韩国总统文在寅在曼谷东盟峰会期间主度呼吁双方会谈，12月正式双方首脑会谈之前，日本在断供5个月之后，宣布小幅度放宽对韩光刻胶管制，整个韩国半导体行业稍稍松了一口气。

2020年3月日本又首次批准向韩国出口EUV光刻胶，并特别允许三星获得至少9个月用量的7/5nm EUV光刻胶，当时三星的7nm逻辑制程已经接近停摆。

面对小日子的断供，韩国采取不卑不亢的软硬兼施的策略，虽然是文在寅事先喊话日本要求会谈，但日本也很好的反馈韩国，也就是双方并没有把事做绝，都留了一线。

这里面当然也有美国的强力斡旋，毕竟属于盟友关系，所以日本当时是开了口子，韩国半导体也得到喘息机会。

最终韩国在2023年提出第三方代偿方案，由韩国政府设立公共赔偿基金向韩国受害劳工赔付，等于是用韩国的钱还日本的债，最终日韩于2023年3月正式和解，日本正式解除对韩半导体材料管制。

正因这段管制期间，韩国政府痛定思痛，举国之力发展半导体材料，三星在2020-21年投资韩国14家材料企业，投入资金超3800亿韩元。

经过四年的努力，韩国光刻胶国产化率提升到15%左右，高纯度HF自给率70%，氟聚酰亚胺则是联合中国供应链把日本供应商打压至30%以下。

这段期间韩国光刻胶经过不断努力突破了日本封锁，成就了东进世美肯以及SK Materials两家本土光刻胶公司。

![](images/8a7b67a3d52bbf49c8de.png)

东进世美肯除了开发了ArFi光刻胶，更是成功开发出EUV光刻胶成为日本三巨头以外的全球少数能做EUV光刻胶的企业。

SK Materials除了ArFi光刻胶也开发出针对3D NAND高厚度KrF光刻胶。

不过在韩国本土光刻胶厂商取得不错的进展同时，我们看到韩国的光刻胶国产化率还是很低只有10～15％，这又是为何？两家韩国光刻胶企业不是都能做ArFi光刻胶了吗？

真正的原因是这两家是开发出了ArFi产品，但有产品不代表好用，文章开头说了，这是一个极度庞大且繁琐的系统工艺，涉及的品类以及配方太多，每一种又必须跟光刻机与涂胶显影设备搭配。

所以即便韩国本土厂商开发出来相关产品，最终fab还是不太愿意使用。

这里还有一个很大的原因，事实上日本只有5个月的时间是真正禁死的，这几个月韩国半导体产出确实暴跌，但同年12月日本开了口子之后，韩国半导体工厂也开始恢复，务实的政治谈判功不可没。

透过2019年日韩的半导体材料摩擦，我们可以清楚地看出所有对应关系与对半导体行业的影响。

目前小日子由比当初安倍更右的高市早苗领导，挑衅意味确实更高，如此一來，我们也很难保持战略定力，该出手教训还是得出手。

可以预见，国内半导体产业尤其是先进制程将承受非常大的压力，光刻胶正是这几年来美国拉着日本荷兰对华封锁笔者最担心的事。

国内半导体行业对此也一直在努力，除了强力攻关，提升国产化率以外，也准备了很多办法，不管台面上还是台面下，你想得到或你想不到的方法，国内无数从业者与贸易商都在看不见的层面上下功夫。

但最终的关键还是日本倒底有没有留口子，跟当初的对韩国一样，没有把事情做绝，这是我们半导体行业是否能正常运转的核心关键。

**国产光刻胶进度如何？**

我想，即便像当初韩国本土光刻胶厂家那般快速，只要没有百分百国产化，哪怕是90%国产，只有10％需要日本，那国内先进制程也可能几个月之后停摆，因为少了10%也不行。

实际上，目前的先进制程光刻胶国产化率不到10％，差距如此之大，谈国产不国产都没太大意义。

整件事的核心还是日本会不会禁死，这才是唯一关键，就跟当初韩国一样，不会也没有其他看点。

至于股票投资，这两天国产光刻胶股当然可以买进，消息面有非常大的助益，发酵越大国产光刻胶能涨越多。

先进制程相关标的可能得先避开，如果日本禁光刻胶这事继续发酵，那最终也会传导到先进制程产能上，必须小心。

这也还没完，每次曝光后的developer还得用显影剂，刻蚀完之后还得用stripper去除残留光刻胶还有清洗液。

上述就是一个多重曝光需要用到光刻胶以及各种化学药剂的过程，而上述的所有化学品，在14nm以下的先进制程中，所有品类目前的国产化率都非常低。

即便门槛较低的显影液或去胶液整体国产化率还不错，但是针对先进制程的国产化率却不到20%，ArFi光刻胶，BARC以及topcoat则更低都在10%以下。

这些正是国产7nm芯片生产过程必须用到的化学品，没有这些化学品，刚刚公布的麒麟9030将没办法生产。

更麻烦的是，一个做先进制程的fab，其前段与后段、所有层，会用到上百种不同的 resists / BARC / topcoat / stripper / developer 。

这也是为什么「替换光刻胶供应商」不是换一桶胶就能完成的原因，因为每个 layer 的相容性、曝光窗口process window、热处理、黏着性、残留物、与光刻机与涂胶显影设备高度绑定。

也就是说用ASML的scanner配套TEL的track，每一个 layer 用的 resists / BARC / topcoat / stripper / developer也是必须是固定vendor与配方，研发能力强的fab甚至会要求vendor出独特的配方，配套自家工艺。

如果今天我们把ASML的scanner换成nikon或者国产光刻机推出，那上述的所有化学品都要有跟着全换一遍相配套的配方。

简单说就是未來如果设备换了，光刻胶与化学品也得跟着换相应的配方。

所以一直有很多人关心国产光刻机进度，却没有人关心光刻制程中还有一大堆要配套的设备与化学品神智还有计算光刻等软件，这是一整套的系统级工艺，每一项缺一不可，如此多环节配套且高度系统化的光刻工艺，确实大大增加了攻关难度。

要明白这事有多严重，首先还是得从光刻胶本身谈起

光刻胶有非常多种，PCB，面板，LED，先进封装都要用到，但这些领域的光刻胶要求不高，主要还是线宽要求不高。

半导体用光刻胶是最高级别，分类主要以曝光的光源波长也可以说光刻机来分

1. G-line (436 nm)、i-line (365 nm)：对应0.25um以上的工艺。

2. KrF (248 nm)：对应0.13~0.25um工艺。

3. ArF (193 nm)

    ArF dry (干式) : 0.11um~65nm工艺

    ArF immersion（浸没式）：45nm~7nm

4. EUV (13.5 nm)：7nm~2nm

从曝光波长也就是光刻机种类来区分，针对不同的波长有不同的光刻胶，这属于光刻胶的大分类。

事实上不论哪种波长的光刻胶，都不是单一种光刻胶，而是每种波长的光刻胶高达数十种甚至上百种。

每种波长光刻胶之下，我们还能依照不同功能来区分

比如正/负胶，line/space 以及 contact-hole 专用配方，topcoat / BARC（底抗反射涂层），显影剂/去胶剂、以及不同硬掩膜（hard mask）材料 等。每一类都有专门配方。

所以光刻胶不是「一个产品」，即便是同一波长，比如A胶他也会有多个用途/多种化学配方/多个 tone 与不同解析度-灵敏度 trade-off 取舍。

**光刻胶用在哪？**

一颗先进制程芯片有大几十层，大致可分为器件层，金属层，介质层，接触孔层，钝化层等。

把这些功能层加工出来，主要就是透过反覆的增材与减材的逐层加工，一层一层把芯片所有功能层做出来，整个过程可能会超过上百层工序。

我们以国内用DUVi光刻机＋SAQP四重曝光制作7nm的工艺为例，由于多重曝光会显着增加 litho-step、mask 、以及所需的材料/处理步骤，最终整个加工步骤在120～130层左右。

其中需要光罩做图型定义的mask layer超40层，硬掩膜hard mask约20层，有些非关键层可以直接用光刻胶做软掩膜，剩下有一半是不需要光刻的沉积层，离子注入层以及平坦化层等等。

也就是说四重曝光的7nm芯片制程，需要用到光刻胶的超60层左右。

这代表制作一颗7nm芯片需要60种光刻胶？

事实上数量不是这么算的，60层有很多是同一个SKU，而一个layer也不一定只用一种光刻胶。

大类来说，这个7nm芯片制作过程需要用到 I胶，K胶，A胶以及浸没式A胶，因为7nm芯片，在高层数金属层也会用到i-line以及KrF这类成熟制程的光刻机。

单单针对ArFi的光刻胶，就多达数十种，vendor必须推出多个不同配方以满足不同的「解析度／灵敏度／线宽/孔径/溶解特性/耐刻蚀性/热稳定性／相容底层」的需求。

比如，不同的正胶 L/S（line/space）配方、接触孔（contact hole）优化配方、极高解析（低 LER）配方、低溶解度配方、低残留配方、双重图形化/间接 patterning 专用配方、以及配合某些硬掩膜的专用配方等。

每一种配方在化学组成，聚合物主体、PAG、溶剂、溶解抑制剂、添加剂等都会不同。

目前全球主要光刻胶供应商，JSR、TOK、信越等对于 ArFi 都有数十款SKU以供选择（按层、按用途、按曝光工具/数据窗）。

而且上述每一家都有自己擅长的配方，比如否种功能用某家的良率会更好，整体来说目前ArFi光刻胶，信越的评价最好，良率最高。

所有的ArFi胶有数十种，那制作一颗7nm芯片需要用到多少种呢？

我们以M1/M2这样的key metal layer用多曝制程来说，这会牵涉到不同的材料/涂层步骤，每一步都可能用不同配方，

首先，第一道光刻（mandrel）用 ArFi 正胶（line/space 配方) ;

刻蚀／形成 spacer，这牵涉到hard mask材料，而hard mask可能用不同的沉积材料/蚀刻配方。

接下来用氧化或氮化层沉积，再做 spacer，然后在用stripper去除 mandrel；

然后再做第二次光刻，如果是LELE，可能要另一种光刻胶或相同配方但不同曝光条件；

上述的一个多重曝光工艺要用到许多的不同功能的光刻胶，但还远不止与此，因为每一次coating涂胶前还得先涂一层BARC底部抗反射涂层。

浸没式光刻制程，coating之后，前烘完还得在光刻胶上涂一层topcoat用来保护光刻胶跟纯水直接接触。

SK Materials除了ArFi光刻胶也开发出针对3D NAND高厚度KrF光刻胶。

不过在韩国本土光刻胶厂商取得不错的进展同时，我们看到韩国的光刻胶国产化率还是很低只有10～15％，这又是为何？两家韩国光刻胶企业不是都能做ArFi光刻胶了吗？

真正的原因是这两家是开发出了ArFi产品，但有产品不代表好用，文章开头说了，这是一个极度庞大且繁琐的系统工艺，涉及的品类以及配方太多，每一种又必须跟光刻机与涂胶显影设备搭配。

所以即便韩国本土厂商开发出来相关产品，最终fab还是不太愿意使用。

这里还有一个很大的原因，事实上日本只有5个月的时间是真正禁死的，这几个月韩国半导体产出确实暴跌，但同年12月日本开了口子之后，韩国半导体工厂也开始恢复，务实的政治谈判功不可没。

透过2019年日韩的半导体材料摩擦，我们可以清楚地看出所有对应关系与对半导体行业的影响。

目前小日子由比当初安倍更右的高市早苗领导，挑衅意味确实更高，如此一來，我们也很难保持战略定力，该出手教训还是得出手。

可以预见，国内半导体产业尤其是先进制程将承受非常大的压力，光刻胶正是这几年来美国拉着日本荷兰对华封锁笔者最担心的事。

国内半导体行业对此也一直在努力，除了强力攻关，提升国产化率以外，也准备了很多办法，不管台面上还是台面下，你想得到或你想不到的方法，国内无数从业者与贸易商都在看不见的层面上下功夫。

但最终的关键还是日本倒底有没有留口子，跟当初的对韩国一样，没有把事情做绝，这是我们半导体行业是否能正常运转的核心关键。

**国产光刻胶进度如何？**

我想，即便像当初韩国本土光刻胶厂家那般快速，只要没有百分百国产化，哪怕是90%国产，只有10％需要日本，那国内先进制程也可能几个月之后停摆，因为少了10%也不行。

实际上，目前的先进制程光刻胶国产化率不到10％，差距如此之大，谈国产不国产都没太大意义。

整件事的核心还是日本会不会禁死，这才是唯一关键，就跟当初韩国一样，不会也没有其他看点。

至于股票投资，这两天国产光刻胶股当然可以买进，消息面有非常大的助益，发酵越大国产光刻胶能涨越多。

先进制程相关标的可能得先避开，如果日本禁光刻胶这事继续发酵，那最终也会传导到先进制程产能上，必须小心。
