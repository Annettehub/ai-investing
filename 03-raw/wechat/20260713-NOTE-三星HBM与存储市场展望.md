# 2026-07-13 三星HBM与存储市场展望：专家访谈纪要

整理自2030FY访谈，受访者为三星专家。访谈时间2026年7月11日。

## 核心要点

1. **A公司传闻**：3,000亿元订单规模过高，真实长协签订集中于Q3。北美四大已签约，中国客户待签。

2. **中国存储企业**：长鑫DRAM产能<20万片/月，HBM处于HBM2阶段，落后主流3-4代。长江存储NAND性能接近国际水平但Xtacking良率偏低成本较高。

3. **三星产能**：DRAM月产能约60万片。平泽四厂2027年起释放产能，总设计20万片/月。

4. **HBM4/E竞争**：三星HBM4采用1c工艺+12nm Base die，竞争力强于1b的海力士/美光。HBM4E阶段美光可能以台积电3nm Base die最强。

5. **价格趋势**：2026年每季度环比增10-15%，2027年涨幅显著放缓。AI服务器存储成本占比从10%升至30%，2027年或超60%。

6. **长协锁定**：三星锁定约70%服务器DRAM产能。美光收到预付款约220亿美元。
