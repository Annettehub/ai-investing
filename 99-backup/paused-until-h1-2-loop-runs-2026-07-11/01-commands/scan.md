# /scan — 全池异动扫描

> 全股票池（持仓+观察）+ 宏观指标一键扫描，快速发现需要关注的信号。

## 执行步骤

### Step 0: 行情快照（自动）
```bash
pipenv run python scripts/fetch_market_data.py --from-watchlist
pipenv run python scripts/fetch_macro_data.py
```
→ 获取全池 11 只最新价格 + VIX/SPY/QQQ/黄金/原油/BTC/DXY

### Step 1: 异动检测
对每只标的检查：
- 涨跌幅 ≥ ±3% → 🔔 异动
- 涨跌幅 ≥ ±5% → 🔴 重大异动
- 连续 3 天同向 → 📈/📉 趋势确认

### Step 2: 持仓扫描（重仓+轻仓，共6只）
对每只持仓标的：
1. 检查是否有异动 → 有则搜索最近新闻/公告/研报
2. 检查 company 文件中的关键指标是否需要更新
3. 检查是否有新财报/业绩预告 → 标出时间
4. 总结：⚠️ 需要关注 or ✅ 正常

### Step 3: 观察标的扫描（5只）
对每只观察标的：
1. 技术面：涨跌幅、量比、均线位置
2. 消息面：搜索最近一周重要新闻（WebSearch）
3. 基本面：是否有财报、重大合同、产品进展
4. 标记：值得跟进 or 继续观察

### Step 4: 产业方向扫描（4个）
对每个产业主题：
1. 搜索该方向最近重要事件（WebSearch）
2. 检查 themes/ 文件中的追踪节点是否有新进展
3. 标记里程碑事件（如有影响观察标的的，特别标注）

### Step 5: 宏观快照
- VIX 是否 >25（恐慌信号）
- DXY 走势对港股影响
- 黄金/原油异常波动
- S&P 500 / 纳指 技术位

### Step 6: 汇总输出
按严重程度排序：
1. 🔴 需要立即关注
2. 🟡 本周注意
3. 🟢 正常

输出到 `04-output/scan-YYYY-MM-DD.md`
