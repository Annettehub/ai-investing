# 规则晋升技能（rule-promotion）

> 能力说明书：从重复模式中提炼规则，验证 ≥3 次后晋升 rules.md。

## 执行流程

1. **模式观察**：在 /today /weekly /research 中发现重复出现的因果模式
2. **记录候选**：在 `02-kb/rules.md` 的「候选规则」中记录
   - 格式：`候选R: [描述] (本周验证次数: n)`
3. **验证计数**：每次出现同类模式，计数 +1
4. **晋升规则**：当 n ≥ 3 时，人工确认后写入「生效规则」
   - 格式：`R编号: [规则] [验证次数] [首次观察] [最近确认] [来源]`
5. **被推翻处理**：规则失效时，移入 `00-system/false-beliefs.md`
   - 格式：`FB编号: [原规则] [推翻日期] [原因] [教训]`

## 候选规则示例

```markdown
候选R: AI 扩产新闻发布后，L2 芯片标的 7 天内平均跑赢大盘 3%（本周验证次数: 2）
```
