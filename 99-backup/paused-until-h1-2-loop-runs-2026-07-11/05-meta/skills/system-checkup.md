# 系统体检技能（system-checkup）

> 能力说明书：定期执行 /update，检测系统健康问题并给出修复建议。

## 检查项

1. **孤岛检测**：哪些 entities/concepts 没有双向链接？
2. **过时规则检测**：rules.md 哪些规则已失效？
3. **上下文膨胀检测**：哪些文件超过 5000 字？
4. **假设僵尸检测**：哪些假设超过 3 个月无变化？
5. **索引一致性**：index.md 与实际文件是否一致？
6. **摩擦日志回顾**：未解决的 friction 有哪些？
7. **技能老化**：哪些 skills 超过 3 个月未使用？

## 输出格式

```markdown
# YYYY-MM-DD 系统体检报告

## 孤岛检测
- 问题：...
- 建议：...

...

...

...
```
