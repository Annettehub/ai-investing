# 快捷入口（commands/）

> 把常用命令做成一页纸入口，方便复制粘贴给 AI。

## 高频命令

| 命令 | 触发语 | 文件 |
|------|--------|------|
| /idea | "我有一个信号" | 01-commands/idea.md |
| /collect | "采集今日信息" | 01-commands/collect.md |
| /ingest | "入库这篇" | 01-commands/ingest.md |
| /research | "研究 [标的]" | 01-commands/research.md |
| /trade | "基于研究做决策" | 01-commands/trade.md |
| /today | "每日扫描" | 01-commands/today.md |
| /weekly | "周复盘" | 01-commands/weekly.md |
| /update | "系统体检" | 01-commands/update.md |

## 使用方式

1. 直接对 WorkBuddy 说命令名称
2. 或打开对应 .md 文件，复制完整 Prompt 发给 AI
3. 每个命令文件包含：使用场景、输入、执行步骤、检查清单、输出格式
